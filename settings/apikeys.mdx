API Keys are required for secure integrations with third-party systems.  

![API Keys Screenshot](/images/settings/image4.png)

## Key Actions
- View existing API keys with creation and last-used dates.  
- Generate new keys when integrating external services.  

### Generate API Key
![Generate API Key Screenshot](/images/settings/image5.png)

- Provide a descriptive name for the key (e.g., `Production API Key`).  
- Save the key securely, as it will be shown only once.
